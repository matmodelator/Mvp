# Roles

## User Roles in MVP

### Guest

User without account.

Permissions:

- View landing page
- Register account

---

### User

User with account.

Permissions:

- Login
- View profiles
- Read public chat
- Open own profile
- Edit profile

Restrictions:

- Cannot write in public chat
- Cannot like profiles
- Cannot send private messages

---

### Member

User with completed profile.

Permissions:

- Write in public chat
- Like profiles
- Get matches
- Publish status
- Appear in recommendations

Restrictions:

- Cannot send private messages

---

### Verified Member

User with confirmed email.

Permissions:

- Send private messages
- Use advanced chat
- Access VIP features
