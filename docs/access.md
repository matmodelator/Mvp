# Access Levels

## Уровни доступа

| Функция | Только аккаунт | Аккаунт + профиль | Аккаунт + профиль + почта |
|---------|----------------|-------------------|---------------------------|
| Войти в систему | Да | Да | Да |
| Смотреть анкеты | Да | Да | Да |
| Читать общий чат | Да | Да | Да |
| Смотреть главную | Да | Да | Да |
| Открывать свой профиль | Да | Да | Да |
| Редактировать профиль | Да | Да | Да |
| Писать в общий чат | Нет | Да | Да |
| Лайкать анкеты | Нет | Да | Да |
| Получать match | Нет | Да | Да |
| Писать в личные сообщения | Нет | Нет | Да |
| Открывать расширенный чат | Нет | Нет | Да |
| VIP-чат / мгновенный доступ | Нет | Нет | Да |
| Публиковать статус | Нет | Да | Да |
| Попадать в рекомендации полноценно | Нет | Да | Да |

---

## Access Levels Logic

Level 1 — Account

- Login
- View profiles
- Read public chat

Level 2 — Account + Profile

- Write in public chat
- Like profiles
- Get matches

Level 3 — Account + Profile + Verified Email

- Send private messages
- Advanced chat
- VIP features
